Task Requirements:

UI Components:

Notes List:
A component displaying all notes. Each note should show a title and a preview of the content.

Note Form:
A form for adding or editing notes. It should include fields for the title and content of the note.

Functionality:

Authentication:
Create the signup and sign in using Firebase.

Add Note:
Users should be able to add a new note.

Edit Note:
Users should be able to edit an existing note.

Delete Note:
Users should be able to delete a note.

Filter/Search Notes:
Implement a search bar to filter notes based on the title or content.

Favorite Feature:
Add a “Favorite” feature for notes. Users should be able to mark a note as a favorite, which displays after they click the favorite filter button.

Styling:

Use basic CSS or a CSS framework (like Tailwind or Bootstrap) to create a clean and simple UI. Make sure the components are responsive.
